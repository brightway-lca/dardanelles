Dardanelles is a web service for the exchange of simple LCI data. It was built for use in the Brightway Life Cycle Assessment framework, but the underlying format uses Datapackages with CSV and JSON files, and should be usable in other contexts.

This is open source software. You can use it to run your server; all data uploaded to this server is publicly accessible.

To upload you need to register; see the registration function code.
